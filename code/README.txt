Radar results can be generated by running the sections of 
runRadar.m. runRadar.m is a script which creates and runs different
instances of the FMCWRadar class with different target and/or
configuration parameters.

OFDM results can be generated by running the sections of runOfdmModel.m.
runOfdmModel.m is a script which creates and runs different instances
of the ofdmModel class with different configuration parameters.

JRC results can be generated by running the sections of runOfdmRadar.m.
runOfdmRadar.m is a script which creates and runs different instances
of the ofdmRadar class with different target and/or configuration
parameters.

Miscellaneous results have also been generated using idealRangeResponse.m
and plotCapacity.m
